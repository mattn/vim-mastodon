# vim-mastodon

Mastodon client for Vim

Currently, work in progress!

## Usage

### toot

```
:Mastodon toot Blah!
```

### timeline

```
:Mastodon timeline
```

### stream

```
:Mastodon stream
```

## Configuration (WIP)

```
let g:mastodon_host = 'mstdn.jp'
let g:mastodon_access_token = 'XXXXXXX'
```

## Requirements

[webapi-vim](https://github.com/mattn/webapi-vim)
